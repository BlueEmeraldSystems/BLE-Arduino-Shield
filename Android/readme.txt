This directory contains an application pair (Android app + Arduino sketch) 

They provide wireless display and control over eight Arduino I/O lines as defined below:

-- Arduino Port line definitions --
Out: Digital pins P5, P6, P7, P8
In: Analog pins A0, A1, A2, A3

-- Installation --
1) Copy and install Application file 'ArduinoBLEShield.apk'.
2) Build and upload 'sketch_Bluegiga_Simple_Arduino.ino'

-- Usage --
1) Turn on Arduino
2) Start Android application
3) Tap the 'Start' button to initiate Bluetooth communication
4) Tap buttons to turn on/off Arduino ouputs
